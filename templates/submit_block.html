Blockchain Hash Challenge

We are constructing a simple block chain. To submit a new block, you must find the nonse string such that sha256(nonse + previous_hash) contains the appropriate number of zeros at the start of the string. The number of zeros required controls the complexity of the challenge.

View Blockchain

View the block chain to find all current blocks in creation order. You can find the previous_hash by looking at the hash of the last created block.

Submit block

Use the provided validate function to find the appropriate nonse. Provide a unique miner name which will be hashed and added to the block, this will keep your identity secure.
